# OWRT_AC_recovery_script



### Скрипт восстанавливает корректную работу OpenWRT при скачках напряжения, пропадании питания роутера и т. п. корректно перезагружая роутер после восстановления подачи питания.

## Установка
Скопировать, вставить в терминал роутера, нажать ввод по необходимости
```
sh <(wget -qO- https://raw.githubusercontent.com/kkkkCampbell/OWRT_AC_recovery_script/refs/heads/main/ac_recovery.sh)
```
